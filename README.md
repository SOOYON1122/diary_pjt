# diary_pjt - Dotori
## 1. 개요
-  블로그에 일기를 쓰고 공유하고 싶은 친구들끼리 교환 일기처럼 공유하고 싶다.
-  특정 목록의 다이어리에서만 설정해놓은 친구들끼리 공유하고 싶다.
- 그런데 교환일기와 같은 서비스를 제공해주는 블로그 사이트가 없다.
